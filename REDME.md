# Gaza Aid Hub

## Problem → Solution
تواجه غزة بعد الحرب تحديات في توزيع المساعدات وعدم معرفة المواطنين بأماكنها ومواعيدها.  
*الحل:* منصة تفاعلية تعرض أنواع المساعدات، مناطق التوزيع، ومواعيدها، وتتيح تقديم طلب المساعدة بسهولة.

## Target Users
- المواطنين المحتاجين للمساعدات الإنسانية.
- الفرق التطوعية والمؤسسات الخيرية.

## Key Features
- جدول ديناميكي للمساعدات مع فلترة حسب المنطقة.
- نموذج طلب مساعدة مع تحقق من الحقول.
- رسالة نجاح بعد الإرسال.
- فيديو توعوي يظهر عند التمرير.
- Carousel للمؤسسات الداعمة.
- قيمنا الإنسانية مع أيقونات Bootstrap
- تصميم متوافق مع الهواتف والتابلت وسطح المكتب.

## Technologies
- HTML5, CSS3, Bootstrap 5
- JavaScript + jQuery
- GitHub (لرفع المشروع)

## How to Run
1. افتح الملف index.html في أي متصفح حديث.
2. جميع الملفات الأخرى موجودة في المجلدات css, js, assets.
